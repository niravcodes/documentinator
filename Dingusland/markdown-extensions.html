Markdown Extensions

Mermaid Diagrams

Documentator supports Mermaid diagrams out of the box. Use the mermaid code fence:

sequenceDiagram participant User participant Docs participant HTML User->>Docs: Write markdown Docs->>HTML: Convert HTML-->>User: View

Code Highlighting

Code blocks are automatically highlighted:

def hello_world():
    print("Hello from Python!")
function hello() {
  console.log("Hello from JavaScript!");
}
