Configuration

Template Customization

You can customize the HTML template by modifying template/index.html. The template uses Mustache syntax and provides these variables:

  • {{title}} - Page title from the first H1 in markdown
  • {{{content}}} - The rendered markdown content
  • {{{sidebar}}} - The generated navigation sidebar

Styling

The default styles are in template/style.css. You can customize:

  • Sidebar width and colors
  • Typography
  • Code block styling
  • Responsive behavior

Example of customizing colors:

:root {
  --sidebar-width: 300px;
  --sidebar-bg: #2c3e50;
  --text-color: #34495e;
  --hover-color: #f0f2f4;
  --active-color: #42b983;
  --folder-color: #7f8c8d;
}

Custom Scripts

You can add custom JavaScript by:

  1. Adding a <script> tag to the template
  2. Creating a scripts folder in your docs
  3. Adding .js files that will be copied to the output
