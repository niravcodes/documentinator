Getting Started

Installation

Install the package globally using npm:

npm install -g documentator

Basic Usage

  1. Create a docs folder in your project
  2. Add markdown files to it
  3. Run one of these commands:
# Build static HTML
documentator build ./docs

# Serve with live reload
documentator serve ./docs

Markdown Features

Code Blocks

const hello = "world";
console.log(hello);

Mermaid Diagrams

You can create diagrams using Mermaid syntax:

graph TD A[Write Markdown] -->|documentator build| B[Generate HTML] B --> C[View Docs] A -->|documentator serve| D[Dev Server] D --> C
